# HOW TO EXECUTE THE DATABASE SETUP

## EASIEST METHOD - Using Batch File:

1. Double-click on: setup-database.bat
2. Enter your PostgreSQL password when prompted
3. Wait for completion

## ALTERNATIVE - Using PowerShell:

1. Open PowerShell in this directory
2. Run: .\setup-database.ps1
3. Enter your PostgreSQL password when prompted

## MANUAL METHOD - If you prefer step-by-step:

1. Open Command Prompt or PowerShell
2. Navigate to this directory
3. Run these commands one by one:

`cmd
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -c "CREATE DATABASE ezcraftcart_db;"
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ezcraftcart_db -f 01-schema-ddl.sql
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ezcraftcart_db -f 02-categories-data.sql
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ezcraftcart_db -f 03-products-data.sql
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ezcraftcart_db -f 04-product-details-data.sql
`

## USING pgAdmin (GUI):

1. Open pgAdmin
2. Connect to your PostgreSQL server
3. Right-click on "Databases" → Create → Database
   - Name: ezcraftcart_db
4. Open Query Tool for ezcraftcart_db
5. Execute each SQL file in order:
   - 01-schema-ddl.sql
   - 02-categories-data.sql
   - 03-products-data.sql
   - 04-product-details-data.sql

## VERIFICATION:

After setup, run this to verify:

`sql
SELECT COUNT(*) as categories FROM categories;
SELECT COUNT(*) as products FROM products;
SELECT * FROM categories WHERE slug = 'perfumes';
SELECT * FROM products WHERE category_id = '7';
`

Expected Results:
- Categories: 7
- Products: 15
- Perfumes category: 1 row
- Perfume products: 3 rows
